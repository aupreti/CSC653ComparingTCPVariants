To run the first experiment:
   ./experiments.sh

To parse and generate graphs for first experiment:
   python parser1.py

To run the second experiment:
   ./experiments2.sh

To parse and generate graphs for second experiment:
   python parser2.py

To run the third experiment:
   ./experiments3.sh

To parse and generate graphs for third experiment:
   python parser3.py

Note the .tcl filed called by each shell script are located in ex1.tcl, ex2.tcl and ex3.tcl
for experiments 1, 2 and 3 respectively.
